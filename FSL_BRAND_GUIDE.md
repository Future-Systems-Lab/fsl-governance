# FSL Brand Guide

## Sovereign Wellness Lexicon

### Required Language

| Use | Never Use |
|-----|-----------|
| Session | Treatment |
| Participant | Patient |
| Wellness support | Diagnosis |
| Sovereign health | Medical care |
| Sovereign Guide | Provider (in UI text) |
| SovereignLedger | ClaimChain (retired brand name) |
| Sovereign wellness | Clinical care |

### Rules

- No diagnosis, disorder, patient, treatment, clinical, or medical advice language in any user-facing text
- Code variables, routes, and database fields may still use "provider" internally -- only UI copy must say "Sovereign Guide"
- Educational disclaimers required on all session, wellness, and health-data pages
- AI-generated content must use the same lexicon (enforced via system prompt)

## Visual Identity

### Colors

| Name | Hex | Usage |
|------|-----|-------|
| Navy (background) | `#030B0F` | Page backgrounds |
| Aqua (accent) | `#00D9FF` | Primary accent, links, highlights |
| Gold | `#D4AF37` | HNT token, premium indicators |
| Mint | `#7FFFD4` | Secondary accent |
| Green | `#00FF88` | Success states, active indicators |
| Red | `#FF4444` | Error states |
| Muted text | `#7A9BA8` | Secondary text |

### Typography

- **Primary:** Chakra Petch (headings, UI labels, monospace elements)
- **Body:** Inter or system sans-serif
- **Headings on landing pages:** Cinzel (serif, for AlchemistForge)
- **Code/technical:** Space Mono or system monospace

### Dark Mode Only

All FSL platforms are dark mode. No light mode variant exists or is planned.

Display payment options in this order on all platforms:

3. Fiat -- Onramper (decentralized fiat aggregator)
4. Sovereign Navigation -- Insurance navigation support via sovereigncoverage@futuresystemslab.io

## Brand Voice

- Sovereignty-first: the participant owns their data, their wallet, their journey
- Educational, not clinical: FSL is a wellness technology platform, not a medical facility
- Jungian depth meets Web3: shadow work, inner child healing, orthomolecular science
- No corporate wellness language: no "holistic solutions," no "comprehensive care plans"
- Direct and grounded: say what it is, not what it sounds like
