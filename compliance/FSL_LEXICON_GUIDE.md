## Section 2 -- Banned Language (Never Use)

The following words and phrases are **prohibited** across all FSL platforms, AI agent outputs, marketing copy, documentation, and stakeholder communications.

| Banned Term | Reason |
|---|---|
| Patient / patients | Clinical framing; use **wellness participant** |
| Diagnosis / diagnose | Implies clinical authority; FSL does not diagnose |
| Disorder / mental disorder | Stigmatizing and clinical; use **imbalance** or **resonance pattern** |
| Treatment / treat | Clinical framing; use **session**, **wellness experience**, or **support** |
| Prescription / prescribe | Implies pharmaceutical authority (exception: insurance context only) |
| Doctor | Clinical hierarchy; use **practitioner** |
| Symptom | Reductive framing; use **resonance pattern** or **signal** |
| Disease / illness | Fear-based language; use **imbalance** or **terrain disruption** |
| Mental illness | Stigmatizing; use **wellness challenge** or **inner terrain disruption** |
| Medication | Pharmaceutical framing; use **plant intelligence** or **orthomolecular protocol** |
| Cure | Implies clinical outcome claims; use **restoration** or **alignment** |
| Medical record | Centralized framing; use **sovereign record** |
| CDC / FDA / WHO (cited as authority) | FSL does not cite regulatory bodies as guiding authorities |
| Healthcare system | Institutional framing; use **wellness ecosystem** |
| Insurance | Reductive; use **wellness coverage navigation** or **coverage** |
| Therapy | Clinical framing; use **session** or **wellness experience** |
| Therapist | Clinical hierarchy; use **practitioner** or **hypnotherapist** |

## Section 4 -- Brand Voice

**FSL brand voice is:**

- **Empowering** -- Every word should make the participant feel capable and in control
- **Sovereign** -- Language reinforces that the participant owns their data, their identity, and their journey
- **Warm** -- Approachable, human, and compassionate
- **Scientific but accessible** -- Grounded in real modalities, explained without jargon walls

**FSL brand voice is NEVER:**

- Clinical -- We are not a hospital. We do not sound like one.
- Sterile -- No cold, detached, institutional tone.
- Fearful -- We never use scare tactics, worst-case framing, or deficit language.
- Paternalistic -- The participant leads. We support.

## Section 5 -- Disclaimer

> FSL platforms provide sovereign wellness support only. FSL is not a substitute for licensed medical care. AI outputs generated across FSL platforms are wellness suggestions, never diagnoses. No FSL platform, practitioner, or AI agent provides medical advice, clinical diagnosis, or pharmaceutical prescriptions. Your data belongs to you alone.

This disclaimer (or an approved variant) must appear on every FSL platform, in every AI agent system prompt, and in all participant-facing documentation.

---

## Section 6 -- Agent Rules

All AI agents operating within the FSL ecosystem must adhere to the following rules:

1. **Reject banned language** -- If a banned term appears in a prompt or generated output, the agent must automatically substitute the approved alternative from the Substitution Guide (Section 3).

2. **Flag violations** -- If a participant or practitioner uses banned language in input, the agent should gently reframe using approved terminology. Example: "It sounds like you're describing a resonance pattern -- let's explore that."

3. **Use power phrases** -- Agents must proactively incorporate Power Phrases (Section 1) into responses. Every agent interaction should reinforce sovereign wellness language.

4. **Never claim clinical authority** -- Agents must never imply they are providing medical advice, diagnosis, or treatment.

5. **Enforce disclaimer** -- When providing wellness suggestions, agents must include or reference the FSL disclaimer.

6. **Escalation protocol** -- If a participant describes an acute crisis, the agent must provide compassionate support and direct them to appropriate emergency resources without using banned clinical language.
